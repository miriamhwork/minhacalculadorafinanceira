# 💰 Calculadora Financeira Mensal

## Descrição

Este projeto foi desenvolvido para o curso **Suporte ao Cliente e Desenvolvedor Web (Python)** do **Programa Match!**, em parceria com a **MasterTech** e **IBM**.

## 🌐 [Acesse a Calculadora](https://minhacalculadorafinanceira.onrender.com/)

## 🛠️ Como Utilizar a Calculadora

1. **Digite sua meta financeira**:
   - Insira o valor da sua meta no primeiro campo (somente números).

2. **Digite quanto você consegue economizar por mês**:
   - Insira o valor que você pode economizar mensalmente (somente números).

3. **Clique no botão "Calcular"**:
   - Após inserir os valores, clique no botão **"Calcular"**.

4. **Visualize o Resultado**:
   - Você será direcionado para uma nova página que mostrará a quantidade de meses necessários para atingir sua meta financeira.

## 📈 Exemplo de Uso

1. **Meta Financeira**: R$10.000
2. **Economia Mensal**: R$500
3. **Resultado**: Aproximadamente 20 meses

Divirta-se planejando suas finanças!
