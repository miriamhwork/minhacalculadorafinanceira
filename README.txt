Projeto Tema 6: Calculadora financeira mensal

Desenvolvido para o curso de Suporte ao Cliente e Desenvolvedor Web (Python) do Programa Match!
Parceria da MasterTech + IBM

Link para a página de teste: https://minhacalculadorafinanceira.onrender.com/

Como utilizar a calculadora:
Digite sua meta financeira no primeiro campo (somente números)
Digite quanto você consegue economizar por mês (somente números)
Clique no botão "Calcular"
Pronto, você será direcionado à uma nova página com a quantidade de meses que levará para você atingir a sua meta.
